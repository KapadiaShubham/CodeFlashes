Event Listeners

Some Common Events

  • click : the user clicks
  • resize : the user resizes the document
  • keydown, keypress, keyup : the user interacts with the keyboard
  • mouseover : mouse is moved over an element
  • load : a resource has finished loading

CallBack for Event Listeners

                            
                                // Syntax
                                window.addEventListener('load', function(event){
                                    console.log('All resources finished loading!');
                                })
                                // Example-1
                                document.getElementsById('button-1 or someID').addEventListener('click', function(){
                                    console.log('you clicked the button!');
                                })
                                // Example-2
                                document.getElementsById('button-2').addEventListener('mouseover', function(){
                                    document.getElementsById('button-2').innerText = 'you hovered over me!';
                                })
                                // Example-3 Toggle theme by button click
                                // Make sure - you should have a css ready for each element as a child of 'theme2' class
                                // So now we have to add that class to our body tag, when the button is clicked
                                doucument.getElementById('theme-button').addEventListener('click', function(){
                                    document.body.classList.toggle('theme2');
                                });

Scope and 'this' variable

The call method

                            
                                function greet(thing) {
                                    console.log(this + " says greetings, " + thing);
                                }
                                greet.call("Cami", "earthlings")
                                //=> Cami says greetings, earthlings

                                // Another Example
                                var person = {
                                    name: "Samantha",
                                    greet: function(thing) {
                                        console.log(this.name + " says greetings, " + thing);
                                    }
                                }
                                person.greet.call(person, 'neighbor');
                                // Samantha says greetings, neighbor

The apply method

                            
                                function greet(thing1, thing2) {
                                    console.log(this + " says greetings, " + thing1);
                                    console.log("But " + thing2 + " doesn't like " + this);
                                }
                                
                                // How we do in 'call' method
                                greet.call('Samantha', 'Maya', 'Angelina');
                                // Samantha says greetings, Maya
                                // But Angelina doesn't like Samantha
                                
                                // Same thing for 'apply' method
                                greet.apply('Samantha', ['Maya', 'Angelina']);
                                // Samantha says greetings, Maya
                                // But Angelina doesn't like Samantha
Difference between call and apply is that
  • For call you need to count your parameters.
  • But, for apply you have and array.

The bind method

                            
                                function greet(thing1, thing2) {
                                    console.log(this + " says greetings, " + thing1);
                                    console.log("But " + thing2 + " doesn't like " + this);
                                }
                                var boundGreeting = greet.bind('Noah'); // the value for 'this' becomes 'Noah'
                                boundGreeting('Christian', 'Ben');
                                // Noah says greetings, Christian
                                // But Ben doesn't like Noah
