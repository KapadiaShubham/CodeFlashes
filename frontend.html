HTTP methods: GET, POST and DELETE

GET: Requests data from a resource

POST: Submits data to be processed to a specified resource

DELETE: Deletes the resource

GET: pulls

POST: creates

DELETE: removes

HTTP Status Codes

* 100-level - indicates that the request was recieved and understood. But, nothing has happened yet.

* 200-level - indicates the action requested by the client was recieved understood, accepted and processed successfully.

* 300-level - Indicate the client must take additional action to complete the request, often used for redirection

* 400-level - The error seems to have been caused by the client

* 500-level - The server failed to fulfill a valid request

* 100-level - hold on

* 200-level - here you go

* 300-level - go away

* 400-level - you messed up

* 500-level - server's messed up

Standard AJAX request


                                // Standard AJAX request to get a response from an API
                                var a = new XMLHttpRequest();
                                a.addEventListener('readystatechange', function(r) {
                                    // r is the captured event
                                    if (r.target.status === 200) { // 200 means successfull response
                                        console.log(r.target.response);
                                    }
                                });
                                a.open('GET', 'https://api.github.com/users/KapadiaShubham');
                                a.send();

Promise states

* Pending: incomplete

* Fulfilled: complete

* Rejected: failed

fetch() - Calling an API by Promises method


                                // Fetch API - a modern way
                                fetch('https://api.github.com/users/KapadiaShubham')
                                    .then(function(r) {
                                        console.log(r.status);
                                        return r.json();
                                    })
                                    .then(function(j) {
                                        console.log(j);
                                    })
